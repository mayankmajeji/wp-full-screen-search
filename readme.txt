== Installation ==

1. Install WP Full Screen Search plugin by uploading the `wp-full-screen-search` directory to the `/wp-content/plugins/` directory.
2. Activate WP Full Screen Search through the `Plugins` menu in WordPress.
3. Change Colors from `Settings > WP Full Screen Search` menu (Optional Step).
4. Navigate to your WordPress website and click on the WordPress search form to see the magic.

== Frequently Asked Questions ==

= How to use WP Full Screen Search on my WordPress website? =

All you need to do is to activate WP Full Screen Search plugin and sit back. It will automatically convert your website's default WordPress search form to full screen search overlay forms. 

= Can I change background and text colors of full screen overlay form? =

Yes. You can easily change background and text colors from plugin's settings page. 

= Can I change placeholder text of full screen search overlay? =

Yes. You can change placeholder text from the settings page.
